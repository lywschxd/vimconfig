# 使用记录
**nredtree 切换目录**
光标放在nredtree 想切换的目标目录，然后执行cd

**配置nvim使用**
ln -s ~/.vim/vimrc ~/.config/nvim/init.vim

**快捷键**

| 快捷键        | 功能                        |
| ------------- | --------------------------- |
| F2            | tagbar                      |
| F3            | nredtree                    |
| SPC h         | 打开vi配置                  |
| SPC f         | 文件相关                    |
| SPC j         | 跳转相关                    |
| SPC s         | 搜索相关                    |
| SPC a r       | 异步打开shell 做事情        |
| SPC f L       | locate 搜索，需要locate命令 |
| nredtree 上cd | 切换光标所在目录            |
